UIowa Web Crawler

  • Python
  • Scrapy
  • Social Network Analysis (SNA)
  • Google Compute Engine
  • Google Buckets

As I was watching an episode of Halt and Catch Fire on Intranet, I got curious about how many nodes/webpages could possibly be under my Universitys' domain. Furthermore, I wanted to visualize how each department and college interacted with each other and as well as analyze their relationship. Fortunately, I was taking a course on Peer-to-Peer Networks that particular semester, so I was able to apply the theoretical concepts I learnt on to my project. To achieve this, I built a Python based concurrent web crawler that implements a Bloom Filter , to crawl through ~ 10 million webpages and scrape the data on each web page using Scrapy library. The end result was definitely fascinating. You can find the code and a detailed pdf on the results and analysis on my github.
